# WebUI 概览

WebUI 主要包含以下页面：

- 概览
- 测速记录
- 客户端
- IP 池
- DNS 更新
- 配置

## 概览

显示系统总体状态：

- 客户端总数
- 在线数量
- IP 池总量
- 最近测速结果
- 统一轮次状态与倒计时

## 测速记录

显示每次测速历史，包括：

- 客户端 ID
- 运营商
- 下载速度
- 平均延迟
- 最低延迟
- 丢包率
- 综合评分

## 客户端

可管理：

- 白名单模式
- 启动参数生成
- 删除客户端
- 允许/禁用连接

## IP 池

支持：

- 为各运营商分别配置 IP 来源
- 预览拉取结果
- 手动添加 IP
- 文本模式批量编辑
- 测速后自动清理

## DNS 更新

显示：

- 各运营商当前汇总 TopN
- 最终写入 DNS 的结果
- 速度、延迟、丢包率、评分
- 手动更新状态
