# IP 池与拉取源

每个运营商维护独立 IP 池：

- 电信
- 联通
- 移动

## 支持的来源

### HTTP API

接口返回可以是：

- 每行一个 IP
- 逗号分隔
- 分号分隔
- 空格分隔

也支持 `IP#备注` 格式，程序会自动去掉 `#` 后面的内容。

### DoH CNAME

输入域名后，会通过：

```text
https://doh.pub/dns-query?name=example.com&type=A
```

获取 A 记录结果并加入候选池。

## 自动清理

启用后，测速轮次收口时会：

1. 汇总该运营商所有客户端结果
2. 仅保留全局 TopN
3. 删除池中非 TopN IP
4. 再按自动拉取源补拉一次
